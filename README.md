## Build APK

1. Clone the repo:

```bash
git clone https://github.com/alexeyavdeenok/GeoTask.git
cd geotask
```

2. Run build:

Linux / Mac:

```bash
./gradlew assembleDebug
```

Windows:

```bash
gradlew assembleDebug
```

3. APK will be here:

```
app/build/outputs/apk/debug/app-debug.apk
```

### Руководство пользователя

Интерактивная карта: В приложении доступна карта с метками ваших локаций. Для их корректного отображения необходимо разрешить доступ к местоположению при первом запуске.

Актуальная погода (API): На главном экране автоматически выводятся данные о температуре и погодных условиях. Информация подгружается в реальном времени через открытый API Open-Meteo.

Управление местами: Список всех сохраненных точек с названиями доступен в разделе «Места» для быстрого просмотра ваших геозон.

## 🔒 Безопасность приложения

### Шифрование данных

Все чувствительные данные (названия задач, описания, GPS координаты, локации и настройки) автоматически шифруются перед сохранением в локальную БД с использованием **AES-256-GCM**. Это означает, что даже если кто-то получит доступ к файлу базы данных, прочитать данные будет невозможно без криптографического ключа.

### Управление ключами

Криптографические ключи хранятся в **Android KeyStore** — защищённом хранилище операционной системы. Ключи недоступны для прямого извлечения и на современных устройствах используют аппаратный криптопроцессор (TEE/StrongBox) для максимальной защиты.

### Приватность данных

- ✅ **Все данные хранятся локально** на устройстве
- ✅ **Геолокация не передаётся** на сервер
- ✅ **Задачи не синхронизируются** в облако
- ✅ **API погоды** получает только запрос о погоде без сохранения истории

### Разрешения

Приложение требует только необходимые разрешения:
- `ACCESS_FINE_LOCATION` — для работы GPS
- `ACCESS_COARSE_LOCATION` — для сотовых вышек

## 💰 Монетизация и публикация

### Модель: Freemium + Реклама

**Бесплатный уровень:**
- Неограниченные задачи и локации
- Базовые уведомления
- Локальное хранилище

**Реклама в бесплатной версии:**
- Неинтрузивные баннеры Google AdMob в нижней части экрана
- Опциональные вознаграждаемые видео (просмотр видео = премиум функция)

**Premium подписка ($1.99/месяц или $14.99/год):**
- Облачная синхронизация между устройствами
- Синхронизация с Google Calendar
- Расширенная история активности (30 дней)
- Совместное редактирование задач
- Без рекламы
- Экспорт в PDF/CSV

### Каналы публикации

1. **Google Play Store** — основной канал, 70% доход разработчику
2. **F-Droid** — открытый маркетплейс для пользователей, ценящих приватность
3. **GitHub Releases** — прямой доступ к APK файлам
